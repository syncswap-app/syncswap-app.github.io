What is SyncSwap?

SyncSwap is a decentralized exchange and liquidity hub for Ethereum ZK rollups, described in the official docs as infrastructure for token swaps, liquidity management, and DeFi access.

How SyncSwap works

  1. Open the appUse the official syncswap.xyz app route and connect a wallet on a supported network.
  2. Choose tokensSelect the token to spend and the token to receive, then enter the amount to load price and route details.
  3. Review executionCheck output, minimum received, price impact, pool route, and network gas before approving.
  4. Confirm on-chainApprove the token if required, submit the swap in your wallet, and wait for the network confirmation.

Before you swap

Routed swaps

The app uses routing across available pools to seek an efficient execution path for the selected trade.

Multi-pool design

Protocol docs describe Classic, Stable, Aqua, and Range pool models for different asset and liquidity conditions.

ZK-rollup focus

Docs list live support for ZKsync Era, Linea, Scroll, Sophon, and Creator Chain; check the app before transacting.

SyncSwap FAQ

What is SyncSwap used for?

SyncSwap is used to swap tokens and access liquidity features on supported Ethereum ZK rollups, with details available in the official documentation.

Is SyncSwap the official app?

This landing page is not the app; the official links page lists syncswap.xyz as the primary website, so verify the domain before connecting a wallet.

Which networks does SyncSwap support?

The docs list ZKsync Era, Linea, Scroll, Sophon, and Creator Chain as supported ZK rollups; availability can change, so confirm inside the app.

Does SyncSwap show slippage and price impact?

Yes. The trading guide describes price impact, minimum received, and route details that should be reviewed before confirming a swap.

Do I need a wallet for SyncSwap?

Yes. Token swaps are wallet-based transactions, so you need a compatible wallet and gas on the selected network.
